# Contributing to CaroCut

## 开发环境

```bash
pnpm install
cp opencode-template.json opencode.json
pnpm dev
```

详见 [完整指南](docs/GUIDE.md)。

## 代码规范

- TypeScript，遵循 ESLint（`pnpm lint`）
- 语义化提交：`feat:` / `fix:` / `docs:` / `refactor:` / `test:` / `chore:`

## Pull Request

1. Fork → 创建分支 → 提交 → Push → 创建 PR
2. PR 检查清单：代码规范、自我审查、文档更新、lint 通过、测试通过

## 许可证

提交 PR 即同意代码按 [MIT License](LICENSE) 授权。Remotion 有独立的商业许可，详见 [NOTICE](NOTICE)。
