# Remotion Setup

Initializes Remotion project from bundled template. Copies pre-configured project from `templates/template-project/`, installs dependencies, and ensures browser availability.

## Incremental Mode

Remotion 项目初始化不支持增量模式。如果项目已存在，跳过。如需安装额外包，直接 npm install。

---

## Manual Setup (if `project_setup` tool unavailable)

```bash
# 1. Copy bundled template
cp -r templates/template-project ./template-project

# 2. Install dependencies
cd template-project
npm install

# 3. Ensure browser
sh scripts/browser_install.sh
```

---

## Extra Packages

The template already includes `@remotion/transitions` (4.0.417) as a built-in dependency. **Do not install it as an extra package.**

Based on features in `manifests/storyboard.yaml`, install additional packages:

| Feature Detected | Package | Install Command |
|------------------|---------|-----------------|
| Lottie animations | `@remotion/lottie` | `npm install @remotion/lottie lottie-web` |
| 3D content | `@remotion/three` | `npm install @remotion/three three @react-three/fiber` |
| Charts/Graphs | `recharts` | `npm install recharts` |
| GIF support | `@remotion/gif` | `npm install @remotion/gif` |
| Noise effects | `@remotion/noise` | `npm install @remotion/noise` |
| Light leaks | `@remotion/light-leaks` | `npm install @remotion/light-leaks` |

### Feature Detection

Scan storyboard.yaml for keywords to determine required packages:

| Keywords | Package Needed |
|----------|----------------|
| "lottie", "animation json" | @remotion/lottie |
| "3d", "three", "webgl" | @remotion/three |
| "chart", "graph", "bar", "line", "pie" | recharts |
| "gif" | @remotion/gif |

## Template Structure

After setup, the project structure:

```
template-project/
  public/                 # Static assets (empty, populated by builder-pipeline)
    images/
    audio/
    videos/
  src/
    components/           # Shared components
      index.ts           # Barrel export
    primitives/           # Pre-built animation primitives (built-in)
      KenBurns.tsx       # Ken Burns pan/zoom effect
      AnimatedText.tsx   # Text entrance/exit animations
      AnimatedChart.tsx  # Chart animation wrapper
      Transition.tsx     # Scene transition effects (uses @remotion/transitions)
      BreathingSpace.tsx # Pacing pauses between content
      SplitScreen.tsx    # Multi-panel layouts
      DynamicBackground.tsx  # Animated backgrounds
      MaskReveal.tsx     # Mask-based reveal effects
      VideoClip.tsx      # Video playback component
    shots/               # Shot implementations
    audio/               # Audio layer components
    lib/                 # Utilities
      constants.ts       # Placeholder (generated by builder-pipeline)
      timing.ts          # Frame calculation utilities
      resourceMap.ts     # Placeholder (generated by builder-pipeline)
    Composition.tsx      # Main composition
    Root.tsx             # Remotion entry
  package.json           # Includes @remotion/transitions (4.0.417)
  remotion.config.ts
  tsconfig.json
```

## Common Issues

| Issue | Cause | Solution |
|-------|-------|----------|
| Download fails | Network issue | Check VPN, retry download |
| npm install fails | Node version mismatch | Ensure Node.js 18+ |
| browser install fails | No permissions | Run with sudo or fix permissions |
| Type check fails | TypeScript version | Ensure TypeScript 5.0+ |

---

## Notes

- **Always use npm as package manager.** Do not use pnpm or bun -- the template's lock file and scripts are designed for npm only
- Always use `sh scripts/browser_install.sh`, not `npx remotion browser ensure`.
- npm install may take several minutes
- Do not modify template files until builder-pipeline generates resource maps
